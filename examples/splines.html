Flot Examples

Simple example with approximated curves. Original points in dataset marked as points on graph.

var data = [[0, 3], [4, 8], [8, 5], [9, 13]];

$.plot($("#placeholder"), [
        {
            data: data,
            points: {
                show: true
            }
        },
        {
            data: data,
            lines: {
                show: true,
                spline: 250
            }
        }
]);
